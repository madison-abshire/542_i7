## Getting started
- Follow [FLask quickstart guide](https://flask.palletsprojects.com/en/stable/quickstart/)
- Run with `flask --app example run -p 5002`

## Key Routes
base route: `http://localhost:5002`
- All existing Sound Transit stations by name
   `/data/stations/existing`
- All planned Sound Transit stations by name
  `/data/stations/future`
